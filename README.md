# SQL-Projects


## Git Commands

### 1 Initialize Git repository

     git init

### 2 Add every files to repo

   git add .

### 3 Add stage specific files


   git add create_tables.sql insert_data.sql select_queries.sql

### 4 Check status before committing:

   git status

### 5 To commit

     git commit -m "Add SQL scripts for tables, inserts, and queries"

### 6 Push to GitHub

     
      git push origin main
